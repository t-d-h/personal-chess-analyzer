# F09 — Frontend (React + TypeScript)

## Behavior
A React + TypeScript frontend that:
1. Accepts a PGN paste or chess.com URL as input.
2. Submits to `POST /api/games`, then polls `GET /api/jobs/:jobId` showing live progress.
3. On completion, fetches `GET /api/games/:gameId/analysis` and renders:
   - An interactive chessboard with move navigation (← →).
   - A move list with classification badges (best / excellent / good / inaccuracy / mistake / blunder / brilliant / book).
   - A line chart eval graph (x = ply, y = win%).
   - Per-player accuracy and move-count stats.

## Scope
- `frontend/` directory.
- No backend changes.
- Runs on port 3000; proxied behind `nginx` in Docker or direct in dev.

```
    main.tsx
    index.css
  package.json
  tsconfig.json
  vite.config.ts
  Dockerfile
```

## UI flow

### Home page
```
┌────────────────────────────────────────┐
│  ♟ Chess Analyzer                       │
│                                        │
│  [ Paste PGN here...              ]    │
│  or                                    │
│  [ https://chess.com/game/...     ]    │
│                                        │
│              [ Analyze ]               │
└────────────────────────────────────────┘
```

Client-side detection: if input matches `chess.com/game/` send as `{ url }`, otherwise send as `{ pgn }`.

### Analysis page (polling state)
```
┌───────────────────────────────────────┐
│  Analyzing... move 14 of 38           │
│  [████████░░░░░░░░░░░░░░░░] 37%       │
└───────────────────────────────────────┘
```
Poll every 1.5s using `useJobPoller`.

## Dockerfile
```dockerfile
FROM node:20-alpine AS build
WORKDIR /app
COPY package*.json .
RUN npm ci
COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
```

`nginx.conf` proxies `/api/` to `api-gateway:8080`.
