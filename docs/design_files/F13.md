# F13: UI Last Move Indicator

## Objective
Visually indicate the last move on the chessboard by highlighting the "from" and "to" squares and rendering a custom arrow between them when navigating through analysis moves.

## Frontend Implementation

### 1. `AnalysisPage.tsx`
- Import the `Chess` class from `chess.js` (already a dependency).
- Compute the `lastMove` object whenever the `currentPly` changes.
  - The `AnalysisMove` object provides the `fenBefore` and `san` of the move that just occurred.
  - Use `const chess = new Chess(fenBefore); const move = chess.move(san);` to compute the exact `from` and `to` algebraic squares (e.g., `"e2"` to `"e4"`).
- Add state or memoized value for `lastMove: { from: string, to: string } | null`.
- Pass this `lastMove` down as a prop to the `Chessboard` component.

### 2. `Chessboard.tsx`
- Update the `ChessboardProps` interface to accept the optional `lastMove` prop.
- Utilize the built-in `customArrows` and `customSquareStyles` properties of `react-chessboard`.
- **Arrows**: If `lastMove` is present, pass `customArrows={[[lastMove.from, lastMove.to]]}` to draw an arrow from the starting square to the landing square.
- **Highlights**: Apply a subtle background color highlight to both the `from` and `to` squares by passing an object to `customSquareStyles` (e.g., `backgroundColor: 'rgba(255, 255, 0, 0.4)'`).
